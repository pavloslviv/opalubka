<br />

== Description ==

Payment online chat SiteHeart provides a stable conversion in the chain: visitors → communications → payments

= SiteHeart Features =
<ol>
<li>
Chat widget enables a customer not only to communicate with the operator while being on a site but also pay for goods or services presented on it with Visa / MasterCard.
</li>
<li>
The chat window has all necessary information and functions that are always ready to use. It allows you to respond quickly in several dialogues and bill customers on time.
</li>
<li>
Admin panel has everything you need to control operators and payments. You can customize Siteheart here, create forms, themes and templates, add card details and payment systems, create departments and grant operators roles.
</li>
<li>
Free mobile application will allow you to chat with visitors and bill them even when you do not have a computer at hand.</li>
<li>
Use the extension to receive sound and visual alerts of new events in SiteHeart. You will receive alerts even when browser is closed!
</li>
<li>
The main advantage of the service is the ability to easily bill customers, receive payment from Visa and MasterCard. Enter your credit card details, Web Money account or your bank account to organize a quick payments acceptance. 
</li>
<li>
SiteHeart allows to rent Call center operators and pay them only for time they communicate with customers
</li>
</ol>
= SiteHeart plug-in for WordPress =

== Installation ==

1. Download and unpack archive to 'wp-content/plugins/' directory inside of WordPress
2. All extracted files should exist in 'wp-content/plugins/siteheart/'
3. Сlick on “Activate” in the admin panel WordPress - > Plugins-> SiteHeart
4. Authorize in siteheart.com
5. Set the widget
